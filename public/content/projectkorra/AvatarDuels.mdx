# AvatarDuels
`In progress`

**A huge plugin I decided to make to actively update [ProjectKorra](https://projectkorra.com) addons and provide other features for the AvatarDuels server.**
The main goal was to create a fully autonomous AI bender that can engage in duels with opponents (even other AI) over large maps:
- Utilises BFS to create a nav-mesh.
- Traverses large areas of terrain with a milestone system that uses Dijkstra's to link together safe areas.
- Manually performs movements and inputs to activate abilities - like a player would - instead of programmatically.
- Intelligently chooses abilities based on health and distance from opponent.
- Performs movements through a scheduling system to run towards, run away, strafe etc.
