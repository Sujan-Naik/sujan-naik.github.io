# This website !
> Developed by myself using Next.js in Typescript with the App router.
>
> It is statically hosted on github and renders MDX dynamically to be easily updated in the future with more work!
>
> Provides an intuitive and visually appealing experience through the use of wrappers of headlessUI components and customised stylesheets.

## Features
- A project showcase area.
- Model files (with the .glb format) are supported to render animations and models.
- Websites are embedded to showcase easily without having to navigate out.
- An easy to navigate project directory.
